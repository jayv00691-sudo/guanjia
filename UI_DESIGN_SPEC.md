# HAO牌 UI 设计规范文档

## 📐 整体设计规范

### 颜色系统
- **主色调**: Indigo (#6366f1) / Purple (#a855f7)
- **背景**: 渐变 `from-slate-50 to-slate-200` (#f8fafc → #e2e8f0)
- **文字主色**: Slate-800 (#1e293b)
- **文字次色**: Slate-500 (#64748b)
- **成功色**: Green-600 (#16a34a)
- **错误色**: Red-500 (#ef4444)

### 字体
- **字体族**: Inter (Google Fonts)
- **标题**: Bold, 20-24px
- **正文**: Regular, 14-16px
- **小字**: 10-12px

### 圆角规范
- **卡片**: 12-16px (rounded-xl, rounded-2xl)
- **按钮**: 8-12px (rounded-lg, rounded-xl)
- **大按钮**: 24px (rounded-3xl)

### 阴影
- **卡片**: `0 8px 32px 0 rgba(31, 38, 135, 0.15)`
- **按钮**: `shadow-lg`, `shadow-xl`
- **导航栏**: `shadow-2xl`

### 玻璃态效果 (Glass Morphism)
```css
background: rgba(255, 255, 255, 0.65);
backdrop-filter: blur(12px);
border: 1px solid rgba(255, 255, 255, 0.5);
```

## 🎨 组件库

### 按钮样式

**主要按钮**:
- 背景: Indigo-600 渐变到 Purple-600
- 文字: 白色, 粗体
- 圆角: 12-16px
- 悬停: 阴影加深, 轻微缩放

**次要按钮**:
- 背景: 白色或 Slate-100
- 文字: Slate-600
- 边框: 1px Slate-200

### 卡片样式 (Glass Panel)
```css
.glass-panel {
  background: rgba(255, 255, 255, 0.65);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(255, 255, 255, 0.5);
  box-shadow: 0 8px 32px 0 rgba(31, 38, 135, 0.15);
  border-radius: 12-16px;
  padding: 16px;
}
```

### 输入框样式
```css
.glass-input {
  background: rgba(255, 255, 255, 0.4);
  border: 1px solid rgba(255, 255, 255, 0.6);
  backdrop-filter: blur(4px);
  border-radius: 8-12px;
  padding: 12px;
}
```

## 🎯 交互效果

### 悬停效果
- 按钮: `hover:scale-[1.02]` (轻微放大)
- 卡片: `hover:border-indigo-300` (边框变色)
- 导航: `hover:text-slate-600` (文字变色)

### 过渡动画
- 通用: `transition-all duration-300`
- 缩放: `transition-transform`
- 颜色: `transition-colors`

### 激活状态
- 导航按钮: 图标上移 `-translate-y-1`
- 文字: 显示标签 (opacity-100)
- 颜色: Indigo-600

## 📝 设计备注

1. **移动优先**: 设计以移动端为主
2. **毛玻璃效果**: 大量使用玻璃态设计，现代感强
3. **渐变背景**: 整体使用渐变背景，增加层次感
4. **圆角设计**: 所有元素都使用较大圆角，柔和友好
5. **颜色系统**: 以 Indigo/Purple 为主色调，统一协调

## 🎨 Figma 设计建议

在 Figma 中创建时，建议：

1. **创建颜色样式**: 将主色调、文字色等创建为颜色样式
2. **组件化**: 将卡片、按钮等创建为组件，方便复用
3. **自动布局**: 使用 Auto Layout 实现响应式布局
4. **设计系统**: 建立完整的设计系统，包含所有组件变体
