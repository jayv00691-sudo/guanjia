# 免费 UI 设计工具推荐

## 🎯 完全免费的工具

### 1. Penpot ⭐ 推荐
- **完全免费**: 开源，无任何限制
- **功能强大**: 类似 Figma 的界面和功能
- **在线使用**: https://penpot.app
- **支持**: 拖拽组件、组件库、自动布局
- **优点**: 
  - 开源，社区活跃
  - 支持导入/导出多种格式
  - 团队协作功能
  - 无需注册即可使用

### 2. Gravit Designer (Corel Vector)
- **免费版**: 功能完整
- **平台**: Web + 桌面版
- **网址**: https://www.corel.com/corel-vector/
- **优点**: 专业级矢量设计工具

### 3. Lunacy
- **免费**: Windows 专用
- **特色**: 支持 Figma 文件导入
- **网址**: https://icons8.com/lunacy
- **优点**: 
  - 本地运行，速度快
  - 可以直接打开 .fig 文件

### 4. Photopea
- **完全免费**: 在线 Photoshop 替代品
- **网址**: https://www.photopea.com
- **优点**: 
  - 支持 PSD 文件
  - 功能强大
  - 无需注册

### 5. Canva
- **免费版**: 基础功能可用
- **网址**: https://www.canva.com
- **优点**: 
  - 模板丰富
  - 操作简单
  - 适合快速设计

## 🚀 快速开始 - Penpot

### 第一步：访问 Penpot
1. 打开浏览器访问：https://penpot.app
2. 点击 "Try Penpot" 或 "Get Started"
3. 可以选择：
   - 在线使用（无需注册）
   - 注册账号（保存设计）

### 第二步：创建新设计
1. 点击 "New Design"
2. 选择画板尺寸（建议：375×812px，iPhone 尺寸）

### 第三步：开始设计
- 使用左侧工具栏创建元素
- 右侧面板调整属性
- 支持拖拽和组件化设计

---

## 📝 Penpot 使用技巧

### 创建颜色样式
1. 选择元素
2. 在右侧面板设置颜色
3. 点击颜色旁边的 "+" 保存为样式

### 创建组件
1. 选中元素
2. 右键 → "Create Component"
3. 可以创建变体

### 自动布局
1. 选中多个元素
2. 右侧面板 → "Layout"
3. 选择方向（水平/垂直）
4. 设置间距

## 💡 其他免费方案

### 方案 1: 使用代码直接设计
如果你熟悉代码，可以直接修改 React 组件来调整 UI，无需设计工具。

- **Figma** (免费版): 个人使用免费，只是功能有限

### 方案 3: 使用模板
- 在 Penpot/Figma 社区找免费模板
- 修改模板适配你的需求
