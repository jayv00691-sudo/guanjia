# Figma 创建指南 - HAO牌 UI 设计

## 🎯 快速开始

### 第一步：创建新文件
1. 登录 Figma (https://www.figma.com)
2. 点击右上角 "New design file"
3. 命名文件为 "HAO牌 UI Design"

---

## 🎨 设置设计系统

### 1. 创建颜色样式 (Color Styles)

在 Figma 中创建以下颜色：

**主色调**:
- `Indigo-500`: `#6366F1`
- `Indigo-600`: `#4F46E5`
- `Purple-500`: `#A855F7`
- `Purple-600`: `#9333EA`

**背景色**:
- `Slate-50`: `#F8FAFC`
- `Slate-200`: `#E2E8F0`
- `Slate-400`: `#94A3B8`
- `Slate-500`: `#64748B`
- `Slate-800`: `#1E293B`

**功能色**:
- `Green-500`: `#22C55E`
- `Green-600`: `#16A34A`
- `Red-500`: `#EF4444`
- `Blue-500`: `#3B82F6`

**创建方法**:
1. 在左侧面板选择 "Design" 标签
2. 点击 "+" 创建新的颜色样式
3. 命名并保存每个颜色

### 2. 创建文字样式 (Text Styles)

**标题样式**:
- 名称: `Heading/Large`
- 字体: Inter (需要安装)
- 大小: 24px
- 字重: Bold
- 颜色: Slate-800

**正文样式**:
- 名称: `Body/Medium`
- 字体: Inter
- 大小: 16px
- 字重: Regular
- 颜色: Slate-800

**小字样式**:
- 名称: `Caption/Small`
- 字体: Inter
- 大小: 12px
- 字重: Regular
- 颜色: Slate-500

**安装 Inter 字体**:
1. 在 Figma 中，选择文字工具
2. 点击字体下拉菜单
3. 搜索 "Inter"
4. 如果没有，点击 "Get more fonts" 安装

### 3. 创建效果样式 (Effect Styles)

**卡片阴影**:
- 名称: `Shadow/Card`
- 类型: Drop Shadow
- X: 0, Y: 8, Blur: 32, Spread: 0
- 颜色: `rgba(31, 38, 135, 0.15)`

**按钮阴影**:
- 名称: `Shadow/Button`
- 类型: Drop Shadow
- X: 0, Y: 4, Blur: 12, Spread: 0
- 颜色: `rgba(0, 0, 0, 0.1)`

## 🧩 创建组件 (Components)

### 1. 顶部导航栏组件

**创建步骤**:
1. 创建 Frame: `375px × 48px`
2. 背景: `rgba(255, 255, 255, 0.7)` + Blur: 12px
3. 添加内容:
   - 左侧: 圆形 Logo (40×40px, 渐变 Indigo-400 到 Purple-500)
   - 中间: 文字 "HAO牌" (Bold, 14px) + "Nice Hand" (9px, uppercase)
   - 右侧: 菜单图标按钮 (24×24px)

**转换为组件**: 选中 Frame → 右键 → "Create Component" (或按 `Ctrl+Alt+K`)

### 2. 统计卡片组件

**创建步骤**:
1. 创建 Frame: `160px × 100px` (或使用 Auto Layout)
2. 背景: 玻璃态效果
   - Fill: `rgba(255, 255, 255, 0.65)`
   - Effects → Background Blur: 12px
   - Border: 1px, `rgba(255, 255, 255, 0.5)`
3. 圆角: 16px
4. 内边距: 16px
5. 内容布局:
   - 左侧: 圆形图标容器 (48×48px)
     - 背景: 对应颜色 (Green/Blue/Indigo/Purple) + 20% 透明度
     - 图标: 24×24px (使用 Figma 图标库或导入)
   - 右侧: 文字
     - 标题: 14px, Slate-500
     - 数值: 24px, Bold, Slate-800

**创建变体**:
- 创建 4 个变体，分别对应 4 种颜色
- 使用 Component Properties 管理颜色和图标

### 3. 战绩卡片组件

**创建步骤**:
1. Frame: `343px × 80px` (使用 Auto Layout)
2. 背景: 玻璃态效果
3. 左侧边框: 4px, Indigo-500
4. 内容:
   - 左侧: 地点 (Bold, 16px) + 日期/盲注 (12px) + 时长 (10px)
   - 右侧: 盈利金额 (18px, Bold, 绿色/红色)

### 4. 底部导航栏组件

**创建步骤**:
1. Frame: `343px × 60px`
2. 背景: `rgba(255, 255, 255, 0.9)` + Blur: 12px
3. 圆角: 24px
4. 使用 Auto Layout (Horizontal, Space between)
5. 导航按钮:
   - 尺寸: 48×48px
   - 图标: 22×22px
   - 文字: 9px, Bold (激活时显示)
6. 中央按钮:
   - 尺寸: 64×64px
   - 位置: 相对上移 -24px
   - 激活状态: Indigo-600 背景 + 白色图标
   - 非激活: 白色背景 + Indigo-600 图标

## 🎨 玻璃态效果实现

在 Figma 中实现玻璃态效果:

1. **背景层**:
   - Fill: `rgba(255, 255, 255, 0.65)`

2. **模糊效果**:
   - Effects → Background Blur: 12px
   - 注意: Figma 的 Background Blur 需要底层有内容才能看到效果

3. **边框**:
   - Border: 1px
   - 颜色: `rgba(255, 255, 255, 0.5)`

4. **阴影**:
   - Drop Shadow: `0 8px 32px rgba(31, 38, 135, 0.15)`

**提示**: 如果 Background Blur 效果不明显，可以:
- 在卡片下方添加一个模糊的矩形作为背景
- 使用 Layer Blur 代替 Background Blur

## 🔧 Figma 技巧

### Auto Layout 使用
- 选中元素 → 右侧面板 → "Auto Layout"
- 设置方向 (Horizontal/Vertical)
- 设置间距 (Gap)
- 设置内边距 (Padding)

### 组件变体 (Variants)
- 创建组件后，可以创建变体
- 使用 Component Properties 管理不同状态
- 例如: 导航按钮的激活/非激活状态

### 约束 (Constraints)
- 设置元素在不同屏幕尺寸下的行为
- 例如: 导航栏固定在顶部

### 原型 (Prototyping)
- 连接不同页面，创建交互流程
- 设置点击区域和过渡动画
